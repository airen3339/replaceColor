 # ColorReplace 图片换色工具

## 产品概述

ColorReplace 是一款专业的图片颜色替换工具，能够精确识别并替换图片中的指定颜色，广泛应用于游戏素材处理、UI设计调色、批量图片处理等场景。

## 核心功能

### 1. 颜色替换
- **精确替换**：支持精确匹配指定颜色进行替换
- **容差替换**：支持设置颜色容差，替换相近颜色
- **批量替换**：支持同时替换多种颜色
- **透明度保持**：替换时可保持原图透明度信息

### 2. 颜色选取
- **取色器**：从图片中直接吸取颜色
- **颜色面板**：支持RGB、HSV等多种颜色模式
- **颜色历史**：记录最近使用的颜色

### 3. 批量处理
- **文件夹处理**：支持整个文件夹批量换色
- **格式转换**：支持多种图片格式输入输出
- **进度显示**：实时显示处理进度

### 4. 图片格式支持
- PNG（支持透明通道）
- JPG/JPEG
- BMP
- GIF

## 使用场景

1. **游戏开发**：批量修改游戏素材颜色，生成不同配色方案
2. **UI设计**：快速调整界面元素配色
3. **图标制作**：生成多种颜色版本的图标
4. **素材处理**：统一调整素材库的色调

### 试用限制
- 试用版：功能无限制，有水印提示
- 注册版：完整功能，无任何限制

## 技术特点

1. **高性能处理**：采用多线程并行处理，大幅提升批量处理速度
2. **内存优化**：智能内存管理，支持处理超大图片
3. **精确算法**：采用专业颜色空间转换算法，确保颜色替换准确
